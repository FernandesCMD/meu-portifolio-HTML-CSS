# meu-portifolio-HTML-CSS
# 💼 Portfólio Pessoal

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido com **HTML5** e **CSS3**.

O projeto foi criado com o objetivo de apresentar minha trajetória, habilidades e projetos, além de aplicar na prática os conhecimentos adquiridos durante meus estudos em Desenvolvimento Web.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox

## ✨ Funcionalidades

* Página inicial com apresentação profissional.
* Seção "Sobre Mim".
* Links para redes sociais em formato de ícones.
* Botão para acesso ao currículo.
* Layout responsivo e organizado.

## 📈 Evolução do projeto

Este portfólio está em constante evolução. Novas funcionalidades e melhorias serão adicionadas conforme avanço nos estudos e concluo novos cursos.
